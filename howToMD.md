````markdown
# Markdown Primer

Markdown is a lightweight markup language used to format text.  
It is widely used on GitHub for README files, documentation, and wikis.

---

## Headings
Use `#` symbols for headings. The number of `#` controls the level.

```markdown
# H1 Heading
## H2 Heading
### H3 Heading
````

# H1 Heading

## H2 Heading

### H3 Heading

---

## Text Formatting

```markdown
*Italic text* or _Italic text_  
**Bold text**  
***Bold and italic***  
~~Strikethrough~~
```

*Italic text* or *Italic text*
**Bold text**
***Bold and italic***
~~Strikethrough~~

---

## Lists

### Unordered List

```markdown
- Item 1
- Item 2
  - Subitem
- Item 3
```

* Item 1
* Item 2

  * Subitem
* Item 3

### Ordered List

```markdown
1. First
2. Second
3. Third
```

1. First
2. Second
3. Third


[OpenAI](https://www.openai.com)
